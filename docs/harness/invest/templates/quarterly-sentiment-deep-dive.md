# 최근 분기 실적·센티먼트 심층 비교 템플릿

## 1. 목적

이 템플릿은 연간 재무 중심 분석이 놓치기 쉬운 최근 변화율, 컨센서스 변화, 어닝콜 톤, 뉴스·수급·애널리스트 리비전을 심층 비교하기 위한 보조 산출물이다.

이벤트 드리븐 분석은 독립 모드가 아니라 이 템플릿 안의 `특정 이벤트 / 촉매` 하위 분석으로 처리한다.

사용 조건:

- 입력 수집 게이트의 `분석 초점`이 `최근 분기 실적·센티먼트 심층형`일 때 필수
- `분석 초점`이 `혼합형`일 때도 요약 버전 필수
- 사용자가 특정 이벤트 / 촉매를 입력하면 이벤트 전후 데이터를 추가 분석

## 2. 데이터 범위

| 항목 | 기본 범위 | 비고 |
|---|---|---|
| 분기 실적 | 최근 4~8개 분기 | 가능하면 8개 분기 |
| 컨센서스 비교 | 최근 4개 분기 | 실제치 vs 컨센서스 |
| 가이던스 | 최근 4개 분기 | 상향/하향/유지 |
| 어닝콜 | 최근 2~4개 콜 | 경영진 톤 변화 |
| 뉴스·센티먼트 | 최근 30일 / 90일 | 실적 전후 분리 |
| 주가 반응 | 실적 발표 전후 D-5, D+1, D+5, D+20 | 가격·거래량 |
| 애널리스트 리비전 | 최근 30일 / 90일 | EPS, 목표가, 투자의견 |
| 피어 비교 | 같은 기간 주요 피어 | 최소 3개, 기본 5개 |
| 특정 이벤트 / 촉매 | 사용자 입력 이벤트 전후 | 예: 실적, 가이던스, M&A, 규제, 소송, 신제품, 관세, 수주, 공급망 |
